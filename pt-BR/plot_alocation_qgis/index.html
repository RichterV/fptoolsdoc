    Alocação de Parcelas com QGIS

    Com o Plugin FPT Plot Allocation, você poderá alocar parcelas de inventário utilizando diversos métodos de amostragem e formatos de parcela, tudo por meio de uma interface gráfica intuitiva e amigável.

    Instalação do Plugin

    No QGIS, vá para "Plugins" "Gerenciar e Instalar Plugins" e pesquise por "FPT Plot Allocation".

    Instalação via arquivo ZIP

    Se preferir, clique aqui para baixar o arquivo ZIP contendo os arquivos do plugin. Para mais informações sobre a instalação, consulte o guia de instalação de plugins do QGIS.

    Funcionalidades do Plugin

    Parâmetros Descrição
    Tipo de distribuição Tipo de distribuição das parcelas.
    Formato da parcela Formato da parcela que será instalada.
    Camada de limite O shapefile com o polígono da área.
    Usar uma coluna para definir o número de parcelas Se selecionado, permite ao usuário escolher uma coluna para definir o número de parcelas por polígono.
    Área da parcela Área em metros quadrados que cada parcela terá.
    Número de parcelas ou porcentagem da amostragem Pode ser o número de parcelas ou a porcentagem da área total que será amostrada. Se valor < 1, será entendido como porcentagem. Se valor >= 1, será entendido que você está definindo a quantidade de parcelas a serem alocadas.
    Por hectare Se selecionado, aplica o valor de Número de parcelas ou porcentagem da amostragem por hectare. O valor deve ser maior que 1.
    Distância mínima da borda (metros) (Opcional) Define a distância mínima em metros que cada parcela terá da borda.
    EPSG O EPSG da sua área.
    Criar buffer para as parcelas (Opcional) Se marcado, criará um buffer mostrando a área das parcelas.
    Saída (Opcional) Selecione uma pasta para salvar o shapefile das suas parcelas.

    Argumentos disponíveis

    • random : Aloca as parcelas de forma aleatória.
    • best sampling : (recomendado) Aloca as parcelas na melhor distribuição possível para a área, considerando os parâmetros estabelecidos.
    • systematic : Aloca as parcelas em uma distribuição em grade, não permite a definição de número de parcelas ou porcentagem da amostragem e aloca todas as parcelas possíveis.
    • systematic custom : Aloca as parcelas em uma distribuição em grade com as distâncias x e y das linhas da grade e o ângulo de rotação da grade definidos pelo usuário.
    • round : Considera que as parcelas terão um formato arredondado.
    • squared : Considera que as parcelas terão um formato quadrado.
    • rectangle : Permite que o usuário insira os tamanhos X e Y do formato retangular desejado.
    • Para obter a melhor precisão nos cálculos, você deve selecionar a zona UTM na qual sua área está localizada. Encontre sua zona UTM.
